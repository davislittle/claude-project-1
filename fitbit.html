Connect your Fitbit.

This tab reads your Fitbit data using the official Fitbit Web API. You'll need a developer app + a backend proxy to handle the token exchange (Fitbit doesn't allow PKCE-only browser flows for all scopes).
  1. Go to dev.fitbit.com → Register an App
  2. Set OAuth 2.0 Application Type to Personal
  3. Set Callback URL to YOUR-SITE-URL/fitbit.html
  4. Copy your Client ID and Client Secret
  5. Deploy a backend function (Netlify / Vercel) that exchanges the auth code for tokens — set FITBIT_CLIENT_ID, FITBIT_CLIENT_SECRET, FITBIT_REDIRECT_URI as env vars and POST to https://api.fitbit.com/oauth2/token
  6. Paste your Client ID + proxy URL below
